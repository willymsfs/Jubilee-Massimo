# The Catholic Jubilee of Hope — *Spes Non Confundit*

An elegant, single-file dynamic HTML presentation based on lectures by **Dr. Massimo Faggioli** (Villanova University) on the Catholic Jubilee of Hope 2025.

## Live Site

**https://willymsfs.github.io/Jubilee-Massimo/**

## Sources

Three lectures by Dr. Massimo Faggioli:

1. **"Celebrate Jubilee"** — Franciscan Renewal Center, Phoenix, AZ
2. **"2024 Paluch Lecture"** — University of Saint Mary of the Lake, Mundelein Seminary, Chicago
3. **"From Pope Francis to Leo XIV: Global Historical Perspectives"** — University College Dublin, Ireland, 2025

## Eight Theological Modules

| # | Module | Key Themes |
|---|--------|-----------|
| 01 | The Jubilee in History | Biblical Jubilee, Crusades ancestry, Boniface VIII (1300), pilgrimage to tourism |
| 02 | Jubilees of the 20th Century | 1950 reconciliation, 1966 Vatican II, 1975 television, Year 2000 |
| 03 | Theology of Hope | *Kataischuno* (Romans 5:5), hope vs. optimism, Walter Benjamin, hope as muscle |
| 04 | Pope Francis and the Jubilee | From Mercy (2015) to Hope (2025), artisan spirituality, synodality, autobiography |
| 05 | Vatican II and the Jubilee | *Gaudium et Spes*, reception crisis, four dimensions of challenge |
| 06 | After Francis: Pope Leo XIV | Pan-American, global, Roman; Lampedusa subtweet; Augustinian unity |
| 07 | Catholicism and the Sense of History | *Historia magistra vitae*, four pathologies, minimum of shared meaning |
| 08 | The Jubilee and Sacred Time | Four modes, expiration time, eschatology, Francis's final legacy |

## Design

- **Fonts:** Playfair Display (headings) · Source Serif 4 (body) · Inter (UI) — 14pt base
- **Palette:** Warm cream-parchment background · gold, navy, crimson, olive accents
- **Architecture:** Single self-contained HTML file — no build step, no dependencies, no broken links
- **Features:** Scroll-reveal animations · responsive layout · module navigation · timeline views · comparison tables · illustrated examples

## Technical

Built as a single `index.html` file (≈ 71KB). Deploys directly to GitHub Pages from the `gh-pages` branch with no build process required.
